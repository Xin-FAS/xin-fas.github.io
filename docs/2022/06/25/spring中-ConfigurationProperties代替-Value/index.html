区别

  1. @Value 只能注入一个属性，@ConfigurationProperties 能匹配多个
  2. @Value 不支持JSR303数据校验，@ConfigurationProperties 配合@Validated 能对数据做校验
  3. @Value 不能对复杂对象的直接封装（对象等）
