介绍

Less（Leaner Style Sheets 的缩写）是一门CSS扩展语言，也成为CSS预处理器。

在CSS的语法基础上，引入了变量，Mixin（混入），运算已经函数等功能，简化了CSS的编写，降低了CSS的维护成本，进入Less中文网站

目前只记录简单使用，主流还是SCSS，详见另一篇

变量命名规则：

  1. 必须有@为前缀
  2. 不能包含特殊字符
  3. 不能以数字开头
  4. 大小写敏感
  5. 必须以;结尾

于SCSS相比：less的变量声明用@，而scss使用$

Less运算

同理，写法和scss是一样的

注意点：

  1. 在运算符的两边最好留一个空格
  2. 做运算时，如果两边只有一个单位，就以这个单位为最后运算结果
  3. 做运算时，如果两边都有单位，则以前面的运算单位为结果
  4. 新版本除法需要使用括号，如：font-size: (320px / @span);
