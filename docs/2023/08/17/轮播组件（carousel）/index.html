轮播列表组件（carousel）

本文最后更新于：2023年8月30日 晚上

需求

  1. 支持水平垂直滚动
  2. 支持动态添加数据项
  3. 可自定义的左右切换箭头
  4. 三种模式可选（无限滚动，循环滚动，按需滚动）

使用

以Vue2为例

需要注意的点：

  1. 轮播的插槽default必须有一个根标签，并且只能有一个
  2. 当使用垂直滚动功能时，父元素一定要存在高度
  3. 一定要指定绑定的key，为空时不会显示当前项目（8/29，待优化）
