  1. bind改为了mounted

  2. unbind改为了unmounted

  3. update改为了updated

设计思路

  1. 使用setInterval操作dom添加光标符号，做出打字效果
  2. 拆分文字，进行一个个的添加，添加时要注意处于打字状态（也就是显示光标）
  3. 要打的文字结束后，关闭控制闪烁的两个定时器，此时关闭后，最后一次还未执行，所以要手动设置一次
  4. 在更新操作时，判断值更新后再重新执行

指令其中的addFlickerText方法，预留了回调形式改变文字，后续可以自定义更多效果（如模拟打错了删几个再打）
